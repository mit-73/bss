# **App Name**: Bilateral Stimulation Station

## Core Features:

- Visual Stimulation Display: Display moving visual stimuli with adjustable directions.
- Auditory Stimulation Playback: Play bilateral auditory stimuli synchronized with visual cues.
- Session Timer Configuration: Allow users to set custom session duration.
- Fullscreen Start with Countdown: Initiate sessions in fullscreen mode with a 3-second countdown.

## Style Guidelines:

- Primary color: Deep calming blue (#3F51B5) for focus and tranquility.
- Background color: Very light desaturated blue (#F0F4F9).
- Accent color: Soft lavender (#9575CD) to highlight controls.
- Body and headline font: 'Inter' sans-serif for clear readability and a neutral feel.
- Simple, minimalist icons for settings and controls.
- Clean and uncluttered layout to minimize distractions.
- Subtle, smooth transitions for visual stimuli movement.
